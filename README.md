## New papers in Aquatic Viruses

### Credits
Antoine Allard.
Check out the [original project](https://github.com/RaunakDey/aquatic_virus_papers/tree/main)

### Description
New papers in Network Science provides a semi-automatically curated list of new publications in Aquatic Viruses. New publications are shared on weekdays on Bluesky ([@networkspapers.sky.social](https://bsky.app/profile/networkspapers.bsky.social)), on Mastodon ([@networkspapers@social.skewed.de](https://social.skewed.de/@networkspapers)), on Twitter ([@networkspapers](https://twitter.com/networkspapers)), via a [mailing list](https://groups.google.com/g/networkspapers), and via a [RSS feed](https://github.com/antoineallard/networkspapers/blob/main/rss.xml) (beta version).

New publications are flagged if their title or abstract (when available) contains any of the following keywords

> Phage, Bacteriophage, aquatic virus, viruses, marine virus, marine viruse, lake virus, marine phages, phage, bacteriophage, bacteriophages

Flagged publications are then manually reviewed prior to being posted to insure broad relevancy.

### Using this code

If you wish to use this code, you will need to add your Twitter Access Keys in the file `config/twitter_cred.json` as

```
{
  "consumer_key"        : "",
  "consumer_secret"     : "",
  "access_token"        : "",
  "access_token_secret" : ""
}
```
